Java 抽象类和接口
发表于 2020-04-27| 更新于 2020-04-27|Java
阅读量:

抽象类

作用：主要起到一个模板的作用，抽象类中的抽象方法必须被子类实现，这样就可以使用一个统一的方法名来调用子类方法。

抽象类无法直接创建对象，必须被继承；抽象方法也必须在子类中被重写。

抽象类中可以没有抽象方法，但即使类中没有抽象方法，抽象类也无法被实例化；反过来，如果一个类中有抽象方法，那么这个类必须声明为抽象类。

抽象方法没有方法体。

关键字：abstract
